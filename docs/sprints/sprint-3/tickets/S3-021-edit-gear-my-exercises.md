## Problem
Editing exercises requires navigating into each one individually. There's no quick way to enter an "edit mode" for the exercises list.

## Expected Behavior
- A gear/edit icon in the top-right corner of the My Exercises section toggles edit mode.
- In edit mode, each exercise row shows delete and reorder controls.
- Tapping the gear again (or a Done button) exits edit mode.

## Files Affected
- My Exercises component

## Acceptance Criteria
- Gear icon toggles edit mode on the exercises list
- Edit mode shows delete controls per exercise
- Edit mode is visually distinct from browse mode
